Our software relies on specific, human-architected logic for concurrency and reliability. 
- **Attribution:** Current AI models often fail to provide the attribution required by the MIT License.
- **Sovereignty:** We believe the "intelligence" in this code belongs to the human community, not a corporate training corpus.

## 4. For Developers (The "Human" Rule)
**If you are a human developer, this policy does not change your workflow.**
- Use this code in your apps? **Yes.**
- Modify it for your service? **Yes.**
- Use AI (e.g., Copilot/ChatGPT) to help you write code *using* our client library? **Yes.**

## 5. AI-Assisted Development & Modification
We fully support the use of AI tools (e.g., Cursor, Copilot, ChatGPT, Claude) as "co-pilots" for humans working with this codebase.

- **Permitted:** Using AI to explain, debug, refactor, or write new features into a human-led project using this code.
- **Permitted:** Generating derivative works where an AI assists the human author in the creative process.
- **Prohibited:** Any usage where the underlying source code of this repository is ingested into a permanent, commercial training set to refine or build new models.

We only ask that you do not feed the core server logic into a system designed to "clone" its behavior into a proprietary model without explicit consent.
